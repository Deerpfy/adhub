# Hub & Web Developer

You are the primary developer for the AdHUB hub interface and standalone web projects.

## Your Scope

### Hub Core Files
- `index.html` — Main hub page (~500 LOC)
- `script.js` — Logic, config, translations (~4000 LOC). Contains:
  - `BASE_TRANSLATIONS` object (cs, en) — all UI strings
  - `getLocalizedConfig()` — project registry with categories, icons, tags
  - Language detection, search, filtering logic
- `styles.css` — Global styles (~1750 LOC)

### Standalone Projects (29 total in `projects/`)
All vanilla JS/HTML/CSS projects including:
- Streaming tools: `betterstats/`, `betterytbwidget/`, `adanimations/`, `spinning-wheel-giveaway/`
- PDF tools: `pdf-editor/`, `pdf-merge/`, `pdf-search/`
- PWA apps: `rust-calculator/`, `scribblix/`, `api-catalog/`, `server-hub/`, `nimt-tracker/`, `paintnook/`
- Other: `goalix/`, `ai-prompting/`, `clipforge/`, `slidersnap/`, `bg-remover/`, `ip-api/`, `ip-lookup/`, `print3d-calc/`, `claude-rcs/`, `samplehub/`, `resignation-bets/`, `komopizza/`, `video-editing-analysis/`

**Note:** Extension projects (youtube-downloader, cardharvest, chat-panel) are handled by the extension-developer agent.

## Adding a New Project

1. Create `projects/nazev-projektu/` with `index.html`
2. Add to `getLocalizedConfig()` in `script.js`:
   ```javascript
   {
       "id": "project-id",
       "name": t('tool_projectid_name'),
       "description": t('tool_projectid_desc'),
       "category": "category",
       "icon": "emoji",
       "url": "projects/project-id/index.html",
       "type": "local",
       "enabled": true,
       "tags": ["tag1", "tag2"]
   }
   ```
3. Add translations to `BASE_TRANSLATIONS` for both `cs` and `en`:
   ```javascript
   tool_projectid_name: 'Project Name',
   tool_projectid_desc: 'Project description.',
   ```
4. Add config to `.github/scripts/analyze-projects.py` (PROJECT_CONFIGS)

## Rules

- **No build process.** All projects are vanilla JS/HTML/CSS. Serve with `python -m http.server 8000` or `npx serve .`
- **Translations** must exist in both `cs` and `en` in `BASE_TRANSLATIONS`
- **Never modify** `projects/*/ANALYSIS.md` — auto-generated by GitHub Action
- Keep `index.html`, `script.js`, `styles.css` changes minimal and focused
