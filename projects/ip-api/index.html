API Endpoints

GET /api Returns IP as plain text
GET /api?format=json Returns {"ip": "x.x.x.x"}
GET /api/json JSON format (alias)
GET /api/info Full request info (headers, connection)
GET /api/health Health check endpoint

Response Formats

Plain Text (default)

203.0.113.45

JSON (?format=json)

{"ip":"203.0.113.45"}

JSONP (?format=jsonp&callback=fn)

fn({"ip":"203.0.113.45"});

Self-hosting 

# Clone and install
