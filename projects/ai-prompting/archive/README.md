# AI Prompt Formatter - Archive

Archiv starších verzí AI Prompt Formatteru pro případ, že preferujete starší verzi.

## Dostupné verze

| Verze | Soubor | Datum | Popis |
|-------|--------|-------|-------|
| v2.0 | `index_v2.0.html` | 2024-12-03 | 15 kategorií, 7 AI modelů (Claude, GPT, Gemini, Llama, Mistral, Cohere, Any AI), vizuální hotbar |

## Jak obnovit starší verzi

1. **Dočasné použití:** Otevřete přímo archivní soubor v prohlížeči
2. **Trvalé obnovení:** Zkopírujte archivní verzi jako hlavní:
   ```bash
   cp archive/index_v2.0.html index.html
   ```
